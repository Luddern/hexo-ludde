hexo架站指令

##Hexo 基本指令語法

  1. 建立部落格：hexo init 你的網站名稱
  2. 開啟伺服器：hexo server
  3. 新增一篇部落格：hexo new “部落格標題”
  4. 輸出實際網頁： hexo generate
  5. 清除快取：hexo clean
  6. 部署網站：hexo deploy
  7. 更新修改push到repo上:hexo clean && hexo deploy
